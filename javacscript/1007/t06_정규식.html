정규식(Regular Expression) 연습

문자열 : 

    형식 : /~~~/플러그인, /[]/플러그인
    플러그인 : g(global) - 전체범위
              m(multi line) - 여러줄에 걸쳐서 처리(textarea)
              s(single line) - 한줄에 걸쳐서 처리(text) 기본값으로 생략 
               //textarea에 입력된 여러 줄의 문자열은 하나의 문자열로 취급되므로, s 플래그 없이도 정규 표현식으로 검색할 수 있습니다.
              i - 영문 대/소문자 구별하지 않을 경우
              u - 유니코드 구별처리시 사용
    정규식을 실행하기 위해서 사용하는 함수 :
      1. 사용자 입력문자열.match(정규식표현) // 문자열이 앞으로
      2. 정규표현식변수.test(사용자 입력문자열) //문자열이 뒤로
        //test >> match : test를 더 많이 씀
